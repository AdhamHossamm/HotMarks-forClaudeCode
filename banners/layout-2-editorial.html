Flat prompts breed flat results
A plugin that teaches Claude Code the difference between a suggestion and an order.
Every prompt you send is treated equally. A throwaway remark and a critical constraint carry the same weight. HotMarks fixes this by letting you mark what matters. Wrap text in **bold** to create a non-negotiable directive. Wrap text in __underline__ to flag critical emphasis. A prompt-submit hook detects your markers and injects semantic context — automatically, on every submit.
